# go-fifa-notes

## 项目简介

`go-fifa-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅用于存放和展示独立的静态页面内容，便于浏览、引用与分享。

## 目录说明

```
go-fifa-notes/
├── index.html          # 主入口页面（可选）
├── pages/              # 存放归档的 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可能包含不同的主题或用途。页面之间无强制关联，均可单独通过浏览器打开或部署到任意静态托管服务中。

- 页面均采用标准 HTML5 格式，兼容主流浏览器。
- 页面内不包含外部追踪、广告或恶意代码。
- 页面资源尽量内联或放置于 `assets/` 目录下，减少外部依赖。

## 使用方式

1. **直接浏览**：克隆或下载仓库后，用浏览器打开 `pages/` 目录下的任意 `.html` 文件。
2. **部署**：可将整个仓库部署至任何静态文件托管服务（如 GitHub Pages、Netlify、Vercel 等）。
3. **修改**：如需新增或修改页面，请遵循现有目录结构，并确保资源路径正确。
